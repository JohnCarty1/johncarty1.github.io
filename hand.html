Gesture-Controlled Robotic Hand

Mechatronics | Embedded Systems | Human-Machine Interaction

I designed and built a gesture-controlled robotic hand as a mechatronics project focused on sensing, actuation, and intuitive system response. The project combined hardware and control logic to create a system that translated user input into physical motion.

Project Overview

The goal of the project was to create a robotic hand capable of responding to gesture-based input in real time. This required integrating mechanical design with electronic control and a clean signal path from user input to actuator output.

Engineering Problem

A central challenge in this project was building a system that felt responsive without becoming unstable or unreliable. Human input is rarely perfectly clean, so the design had to account for signal noise, calibration, and the mechanical limitations of the hand itself. The project required thinking about response speed, repeatability, and the physical realism of the resulting motion.

Technical Work
