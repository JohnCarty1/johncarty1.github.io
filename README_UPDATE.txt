PROFESSIONAL PORTFOLIO UPDATE
=============================

The complete website folder in this package is ready to use as-is.

For a drop-in update to an existing copy of the site:

1. Replace these files in the main website folder:
   - index.html
   - style.css
   - plasma.html
   - orbits.html
   - robotics.html
   - hand.html
   - reaction.html

2. Copy these folders into the main website folder:
   - orbits_documentation
   - orbit_assets

3. Keep the existing media and model files in the main website folder, including:
   - cathode photos/
   - cathode_screenshots/
   - first_plasma_axial.mp4
   - first_plasma_axial_poster.jpg
   - bot_cad.glb
   - hand_model.glb
   - all existing photos, videos, phase_space.gif, and space.js

MAIN DESIGN CHANGES
-------------------
- Every project page now uses the same centered 1080–1140 px content grid.
- Paragraphs remain at a readable width instead of stretching on large monitors.
- All project pages use the same four-card Role / System / Milestone / Tools summary.
- Standalone figures are centered and split layouts use consistent column widths.
- Biomimetic project photos use their full image rather than fixed-height zoomed crops.
- The orbit page now uses the supplied real simulation graphs, which open at full resolution when clicked.
- Desktop, laptop, tablet, and phone layouts share the same hierarchy and collapse without horizontal overflow.
- The reaction-wheel page was rebuilt to match the rest of the portfolio and no longer contains placeholder text.

The commissioning-result section on plasma.html was preserved while being aligned to the shared layout system.
