Overview

Building outward from the dynamics

Reaction wheels change spacecraft orientation by exchanging angular momentum with the body. Accelerating the wheel produces an equal and opposite response in the surrounding structure, allowing attitude changes without expelling propellant.

I began the project for my Analytical Dynamics final presentation, using Euler's equations, the inertia tensor, and conservation of angular momentum to simulate rotational motion. The current goal is to carry that work into a measured physical system and compare the prototype's response with the numerical model.

Simulation Foundation

From rigid-body motion to reaction-wheel control

The model explores how inertia distribution, initial rotation, wheel torque, and wheel speed affect the body's response. These animations are simulation visualizations from the dynamics work, rather than experimental hardware data.

Current Status

An initial hardware platform

Since the original simulation work, I designed and 3D-printed a first reaction-wheel testbed and integrated an IMU with preliminary control electronics. The hardware is still early, but it establishes the physical platform needed for sensor calibration, open-loop characterization, and eventual feedback control.

Takeaway

This project remains in active development. I am keeping the first hardware scope intentionally focused: a clean, measurable one-axis system that connects the rotational dynamics I studied with real sensor data, actuator limits, and control.
